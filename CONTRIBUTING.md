# Contributing to abfool VPS Orchestration

## Quick Start

1. Fork → Clone → Branch (`feat/your-feature`)
2. Make changes
3. Test: `bash abfool-vps help`
4. Commit: `feat(scope): description` (Conventional Commits)
5. Push → PR

## Code Style

- Bash: 4-space indent, `set -euo pipefail`
- Quote all variables: `"${var}"`
- Use functions, add comments
- Test on both Linux and Termux

## Areas

- 🐛 Bug fixes
- 🌐 New provider integrations
- 📝 Documentation
- 🧪 Tests
- 🎨 Web dashboard
